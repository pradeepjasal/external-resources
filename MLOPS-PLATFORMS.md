# MLOps Platforms & Experiment Management

External platforms and utilities that support the operational lifecycle of machine learning systems.  
These tools assist with experiment tracking, model and artifact management, versioning, and reproducibility.

---

## Tools

---

### AIM
AIM is an open-source, self-hosted machine learning experiment tracking system.  
It enables logging of metrics, hyperparameters, images, artifacts, and provides a flexible and fast UI for interactive exploration of experiment results. AIM integrates smoothly with existing training scripts without requiring modification of ML frameworks.

- **URL:** https://github.com/aimhubio/aim
- **License:** Apache 2.0
- **Operating system:** Cross-platform
- **Programming language(s):** Python

---

### DVC
DVC is an open-source tool for versioning datasets and managing machine learning pipelines, enabling reproducible experiments and data provenance tracking.

- **URL:** https://github.com/iterative/dvc
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

### MLflow
MLflow is a platform for managing the end-to-end machine learning lifecycle, including experiment tracking, model registry, and reproducible runs.

- **URL:** https://github.com/mlflow/mlflow
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python, JavaScript

---
### wandb
(Weights & Biases)
Weights & Biases (wandb) is a tool for tracking machine learning experiments, visualizing metrics, and collaborating on model development.

- **URL:** https://github.com/wandb/wandb
- **License:** MIT
- **Operating system:** Unix
- **Programming language(s):** Python
