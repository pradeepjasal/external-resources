# Federated Learning Frameworks

External frameworks that support decentralized or privacy-preserving training of machine learning models without requiring data to be centralized.  
These tools enable collaborative learning across multiple devices, institutions, or edge nodes while preserving data locality and confidentiality.

---

## Tools

---

### Flower Framework
Flower is a flexible, open-source federated learning framework that enables training machine learning models on distributed data sources.  
It supports a wide range of client environments (mobile, edge, cloud), integrates well with common ML libraries, and allows customization of training and aggregation strategies.

- **URL:** https://flower.ai/docs/
- **License:** Apache 2.0
- **Operating system:** Cross-platform
- **Programming language(s):** Python
