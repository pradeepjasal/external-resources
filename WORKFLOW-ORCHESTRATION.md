# Workflow & Orchestration

External tools that define, schedule, and manage multi-step computational workflows and pipelines in a reliable, reproducible way.

---

## Tools

---

### Apache Airflow
Apache Airflow is a platform to programmatically author, schedule, and monitor workflows defined as directed acyclic graphs (DAGs).

- **URL:** https://airflow.apache.org/docs/
- **License:** Apache 2.0
- **Operating system:** Cross-platform
- **Programming language(s):** Python

### Snakemake
Snakemake is a workflow management system enabling reproducible and scalable data analyses using a Python-based rule language.

- **URL:** https://snakemake.readthedocs.io/en/stable/
- **License:** MIT
- **Operating system:** Cross-platform
- **Programming language(s):** Python

---

### Autosubmit
Autosubmit is a lightweight workflow manager designed to meet climate research necessities. Unlike other workflow solutions in the domain, it integrates the capabilities of an experiment manager, workflow orchestrator and monitor in a self-contained application.

- **URL:** https://github.com/BSC-ES/autosubmit
- **License:** GPL-3.0
- **Operating system:** Unix/MacOS
- **Programming language(s):** Python

---

### Autosubmit API
An open-source Python web application that aims to monitor, analyze, and control workflows created and managed with the Autosubmit workflow manager

- **URL:** https://github.com/BSC-ES/autosubmit-api
- **License:** GPL-3.0
- **Operating system:** Unix/MacOS
- **Programming language(s):** Python

---

### Autosubmit GUI
Front-end software that aims to provide users with simplified information from the workflow execution of scientific experiments managed by Autosubmit

- **URL:** https://github.com/BSC-ES/autosubmit-gui
- **License:** MIT
- **Operating system:** Unix/MacOS
- **Programming language(s):** Javascript

---

### NF-core
NF-core is a community-driven curated collection of high-quality, production-ready bioinformatics workflows built using Nextflow. All pipelines follow strict guidelines for portability, reproducibility and best practices across HPC, cloud and containers.

- **URL:** https://github.com/nf-core/
- **License:** MIT
- **Operating system:** Linux / macOS
- **Programming language(s):** Nextflow DSL
