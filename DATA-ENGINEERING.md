### Xarray
Xarray provides labeled multidimensional arrays built on top of NumPy, enabling powerful processing of climate and scientific datasets. Widely used in ML data pipelines.

- **URL:** https://docs.xarray.dev/en/stable/
- **License:** Apache 2.0
- **Operating system:** Cross-platform
- **Programming language(s):** Python

---

### CCrawl Processing Pipeline
The CCrawl processing pipeline is a download and preprocessing toolchain for Common Crawl dumps, preparing large-scale text corpora for LLM training.

- **URL:** https://github.com/BILGEM-NLP-Team/CC_Processing_Pipeline
- **License:** Apache 2.0
- **Operating system:** Unix/MacOS/Windows
- **Programming language(s):** Python

### NVIDIA NeMo Curator
NVIDIA NeMo Curator provides scalable pipelines for cleaning, filtering, and augmenting text, image, and video datasets for generative AI training.

- **URL:** https://github.com/NVIDIA-NeMo/Curator
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

### Anemoi-datasets
Anemoi-datasets is a Python package for representing and accessing Zarr-based datasets optimized for data-driven weather forecasting models.

- **URL:** https://github.com/ecmwf/anemoi-datasets
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

---

### Earthkit-data
earthkit-data is a format-agnostic Python interface for working with meteorological and climate data, integrating various storage formats behind a unified API.

- **URL:** https://github.com/ecmwf/earthkit-data
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

---

### Earthkit-regrid
earthkit-regrid is a Python library for regridding geospatial data, interoperable with the broader Earthkit ecosystem.

- **URL:** https://github.com/ecmwf/earthkit-regrid
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

---

### Earthkit-transforms
earthkit-transforms provides transformation utilities (e.g., temporal aggregation and statistics) for meteorological and climate datasets.

- **URL:** https://github.com/ecmwf/earthkit-transforms
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

### TensorStore
TensorStore is a library for efficiently reading and writing large multi-dimensional arrays, supporting cloud object storage and parallel I/O.

- **URL:** https://github.com/google/tensorstore
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python, C++

---

### Hugging Face Datasets
Hugging Face Datasets is a library and hub for accessing, streaming, and processing large-scale datasets for machine learning and LLM training.

- **URL:** https://github.com/huggingface/datasets
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

### ChromaDB
ChromaDB is an open-source embedding database and vector store that supports hybrid search (vector, full-text, and metadata) for retrieval-augmented generation and analytics.

- **URL:** https://github.com/chroma-core/chroma
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Rust, Python

### Xarray-TensorStore
Xarray-TensorStore allows opening Zarr arrays into Xarray via TensorStore, providing a faster backend for certain large-scale workloads.

- **URL:** https://github.com/google/xarray-tensorstore
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python

Xarray-Beam is a library for building Apache Beam pipelines over Xarray datasets, enabling scalable processing of multidimensional scientific data.

- **URL:** https://github.com/google/xarray-beam
- **License:** Apache 2.0
- **Operating system:** Unix
- **Programming language(s):** Python
