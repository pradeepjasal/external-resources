# BSC AI Factory — External Resources

This repository provides a curated list of **external** software tools, libraries, guides, and reference materials relevant to the activities of the **BSC AI Factory**.

These resources include:

- External documentation and best-practice guides  
- Open-source tools and frameworks used across AI, HPC, and data engineering  
- Public examples, templates, and learning materials  
- Links to community-maintained projects considered useful for development

No internal or proprietary BSC material is stored in this repository.  
All content consists exclusively of **external, publicly accessible resources** selected by the AI Factory team.
