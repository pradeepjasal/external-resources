### Rbbt Synthetic Cancer Genome Workflow
Rbbt SyntheticCancerGenome provides workflows to generate synthetic cancer genomes for benchmarking and testing genomics pipelines. It supports simulation of single nucleotide variants, structural variants, copy number alterations, and clonal evolution patterns, enabling reproducible evaluation of variant calling and analysis workflows.

- **URL:** https://github.com/Rbbt-Workflows/SyntheticCancerGenome
- **License:** MIT
- **Operating system:** Linux / macOS
- **Programming language(s):** Ruby
