# Benchmarks & Evaluation

External frameworks and suites used to evaluate AI models, compare performance across standardized tasks or datasets, and support rigorous assessment of scientific and machine learning workflows.

---

## Tools

---

### ClimaBench / AtmoRep Benchmarking Suite
ClimaBench is a benchmarking suite designed for evaluating climate-oriented machine learning models.  
It provides standardized tasks, datasets, and metrics to measure the performance and credibility of ML models in climate science.  
It is closely related to AtmoRep, an autoregressive climate emulator used for downstream forecasting and scientific evaluation.

- **License:** AGPL-3.0
- **Operating system:** Cross-platform
- **Programming language(s):** Python

---

### EleutherAI LM Evaluation Harness
The LM Evaluation Harness from EleutherAI is a framework for evaluating language models on a large collection of standardized NLP and reasoning benchmarks.

- **URL:** https://github.com/EleutherAI/lm-evaluation-harness
- **License:** MIT
- **Operating system:** Cross-platform
- **Programming language(s):** Python

---

### BigCode Evaluation Harness
The BigCode Evaluation Harness provides tools and benchmarks for systematically evaluating code generation and code understanding models.

- **URL:** https://github.com/bigcode-project/bigcode-evaluation-harness
- **License:** Apache 2.0
- **Operating system:** Linux
- **Programming language(s):** Python
